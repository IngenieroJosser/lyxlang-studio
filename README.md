# LyxLang Studio

**Desarrolla. Colabora. Gana.**  
El nuevo espacio de creación para developers.  
Programa, comparte y crece en una plataforma que fusiona código, comunidad y recompensas.  
100% en el navegador, 100% tuya.

![LyxLang Banner](./public/lyxlang-lyxlang-studio-with-text-removebg-preview.png)

---

##  Características principales

- **Editor Inteligente y Transparente** — Entorno rápido, minimalista y listo para TypeScript, JavaScript y más.  
- **Conecta tu Repositorio** — Importa proyectos desde GitHub o GitLab y colabora en tiempo real.  
- **Sistema de Recompensas** — Desbloquea insignias y logros por tus commits y colaboraciones.  
- **Terminal y Ejecución en Vivo** — Compila y depura sin salir del entorno.  
- **Rendimiento Brutal** — Construido con **Next.js**, **React**, **WebAssembly** y **monaco-editor**.

##  Instalación y uso local

```bash
# Clonar el repositorio
git clone https://github.com/IngenieroJosser/lyxlang-studio.git

# Entrar al directorio
cd lyxlang-studio


```bash
npm run dev
# or
yarn dev
# or
