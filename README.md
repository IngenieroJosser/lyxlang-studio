## ✨ Características principales

- **Editor Inteligente y Transparente** — Rápido, minimalista y optimizado para TypeScript, JavaScript y más.  
- **Conecta tu Repositorio** — Importa proyectos desde GitHub o GitLab y colabora en tiempo real.  
- **Sistema de Recompensas** — Gana insignias y logros por tus commits, aportes y colaboraciones.  
- **Terminal y Ejecución en Vivo** — Compila, ejecuta y depura sin salir del entorno.  
- **Rendimiento Brutal** — Construido con **Next.js**, **React**, **WebAssembly** y **monaco-editor**.  

---

## Filosofía del proyecto

LyxLang Studio no es solo un entorno de desarrollo:  
es un **ecosistema donde el código se convierte en comunidad**,  
y donde cada developer tiene el poder de **crear, colaborar y crecer sin límites**.

## Instalación local

```bash
# Clonar el repositorio
git clone https://github.com/IngenieroJosser/lyxlang-studio.git

# Entrar al directorio del proyecto
cd lyxlang-studio

# Instalar dependencias
npm install
# o
yarn
# o
pnpm install
# o
bun install

# Ejecutar el entorno de desarrollo
npm run dev
# o
yarn dev
# o
pnpm dev
# o
bun dev
